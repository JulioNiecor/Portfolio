# Julio Nieto | Frontend Developer Portfolio

Un portfolio web altamente interactivo y optimizado, diseñado para mostrar proyectos, habilidades y experiencia con una estética premium inspirada en el Glassmorphism y la fluidez 3D. 

Este proyecto ha sido auditado estructuralmente y ha conseguido una calificación técnica de **100/100** bajo las estrictas directrices de React Doctor y WCAG 2.2.

## 🚀 Tecnologías Principales

- **Framework Core:** [Next.js 16.2.6 (App Router)](https://nextjs.org) optimizado para React 19 y Server Components puros.
- **Estilos y Arquitectura CSS:** [Tailwind CSS v4](https://tailwindcss.com) moderno y depurado.
- **Animaciones y Físicas:** [Framer Motion](https://www.framer.com/motion/) con aceleración por GPU.
- **Tipografía e Iconografía:** Geist Font y [Lucide React](https://lucide.dev/)

## ✨ Características Técnicas (Puntuación 100/100)

- **React 19 & Compiler Ready:** Arquitectura moderna que prescinde de memoizaciones manuales obsoletas (`useCallback`, `memo`), delegando la optimización nativa al motor de compilación de React para lograr cero renderizados innecesarios.
- **Seguridad y Rendimiento:** Base de dependencias parcheada y actualizada. Generación de sitios estáticos (SSG) de ultra-baja latencia y carga de imágenes en formato WebP con políticas de retención de diseño (evitando Cumulative Layout Shift).
- **Animaciones de Alto Rendimiento:** Transiciones fluidas a 60 FPS, carruseles 3D con cálculos elásticos (Spring Physics) e iluminación atmosférica matemática que se procesa a nivel de hardware.
- **Accesibilidad Inclusiva (A11y):** Cumplimiento total de las pautas WCAG 2.2, incorporando navegación mediante teclado (`<Link>` semánticos), botones securizados (`type="button"`) y etiquetas legibles para lectores de pantalla (`aria-labels`).
- **Glassmorphism Reactivo:** Implementación de tarjetas de cristal que reaccionan matemáticamente a la posición del puntero, calculando gradientes de forma nativa para no saturar el Virtual DOM de React.

## 💻 Ejecución Local

1. Instala las dependencias del proyecto de forma local:
   ```bash
   npm install
   ```

2. Arranca el servidor local de desarrollo:
   ```bash
   npm run dev
   ```

3. Abre [http://localhost:3000](http://localhost:3000) en tu navegador para interactuar con la interfaz.
